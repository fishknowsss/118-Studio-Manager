# 今日页下半区双栏改版设计

## 目标

把首页“今日”页下半区从“左侧上下堆叠任务池/人员 + 右侧日历”改成“任务池与人员左右分区 + 最右日历独立”的结构。

## 结构

- 上半区保持不变：
  - 日期头部
  - 今日焦点
  - 焦点项目卡片
- 下半区改为三列：
  - 左列：任务池
  - 中列：人员区
  - 右列：迷你日历

## 人员区

- 人员区改为块状小卡片，不再使用行列表。
- 固定预留 `3 x 5` 共 `15` 个卡片槽位。
- 已有人员显示卡片：
  - 姓名
  - 性别
  - 技能标签
  - 任务数
- 未占用槽位显示预留占位。

## 拖拽交互

- 任务拖到人员卡片：
  - 将该任务分配给对应人员
- 人员卡片拖到任务行：
  - 将该任务分配给对应人员
- 空白区域不定义新语义
- 拖拽反馈保持状态驱动：
  - 被悬停的人员卡片高亮
  - 被悬停的任务行高亮

## 实现边界

- 仅修改首页“今日”页下半区结构与交互
- 不改数据库结构
- 复用现有 `assignTaskToPerson`
- 优先复用已有 selector 的人员展示数据

## 自检

- 不做套娃卡片
- 分区以留白和对齐为主，不堆边框
- 中文文案保持短句
